Miért a ProfessionBot?

Modern technológiák és biztonságos működés a sikeres pályázatokért.

Firefox Playwright

GeckoDriver alapú böngésző automatizáció, amely minimalizálja a bot detektálás esélyét és természetes böngészést szimulál.

Smart Checkbox Kezelés

Automatikusan kijelöli az önéletrajzokat és a kötelező GDPR nyilatkozatokat, de okosan kihagyja a marketing célú megkereséseket.

Session Mentés

Sikeres bejelentkezés után a cookie-kat elmenti, így a következő futások alkalmával nem igényel újbóli belépést.

Biztonságos Működés

Az adataid és a jelszavad kizárólag a te helyi gépeden, a config.json fájlban tárolódnak. Semmit sem küldünk külső szerverre.

Dry-Run Mód

Szimulált futás: tesztelheted a működést, a bot kitölt mindent, de nem küldi el ténylegesen a jelentkezést.

Guided Üzemmód

Dönts te minden egyes állásról a terminálban egyetlen gombnyomással (y/n/skip) a teljes kontroll érdekében.

Gyors Kezdés

1 Telepítés

Klónozd a tárolót és telepítsd a szükséges csomagokat:

git clone https://github.com/domedav/profession-bot.git cd profession-bot pip install -r requirements.txt playwright install firefox

2 Konfigurálás

Indítsd el a beállító varázslót a fiókod és bérigényed megadásához:

python3 autoapply.py --config

3 Futtatás

Futtasd a botot számodra megfelelő üzemmódban:

# Guided mód manuális jóváhagyással python3 autoapply.py --guided # Vagy biztonságos tesztelés jelentkezés nélkül python3 autoapply.py --dry-run
